Interface DelegateRenderScheduler<TExecution, TOptions>

A render scheduler that schedules rendering by another one.

The target scheduler can be switched at any time.

Can be constructed by newDelegateRenderScheduler function.

Type parameters

  • TExecution: RenderExecution = RenderExecution

    A type of supported render shot execution context.

  • TOptions: RenderScheduleOptions = RenderScheduleOptions

    A type of accepted render schedule options.
